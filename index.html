OS.puppy.CPUSchSimulator

CPU scheduling and memory allocation simulator

Download this project as a .zip file Download this project as a tar.gz file

--README-- This program simulates the CPU and memory scheduling process, this work was done by Wislay and me(http://chentingpc.me), he was responsible for the UI and I were responsible for the scheduling.

The code is under the BSD protocol, the code is written during 2012-6-22 ~ 2012-6-24.

--EFFECT-- 用户界面 内存区表示内存的使用情况，不同的颜色属于不同的进程， 右边可以输入代码，所需内存单元数，优先级。

指令类型 CALC 运算， RDLK 读锁， WTLK 写锁， RDFL 读文件， WTFL 写文件， URLK 去读锁。 还可以产生中断。

正常调度： 略

中断： 略

有锁调度（读读、读写、写读、写写）： 略

死锁产生： 进程一执行代码： WTLK 0 CALC 10 WTLK 1 UWLK 0 UWLK 1 CALC 5 运行二执行代码： WTLK 1 CALC 10 WTLK 0 UWLK 1 UWLK 0 CALC 5
